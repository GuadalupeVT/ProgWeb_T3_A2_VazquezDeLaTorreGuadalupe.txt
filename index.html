Actividad 2 - Ejercicios JAVASCRIPT [DOM]

Seleccione una opción...

1) Crear un programa en JavaScript que lea los datos necesarios para obtener y mostrar el CURP de una persona.
2) Crear un programa en JavaScript que lea un numero en decimal y lo convierta a binario, octal y hexadecimal.
3) Crear un programa en Javascript que lea una cadena de caracteres y muestre lo siguiente:
a) Cantidad de vocales
b) Cantidad de Consonantes
c) Cantidad de aparación de una letra ingresada por el usuario
d) La cadena como cadena loca, ejemplo: InGeNiErIa
